# Compile instructions
- the latest combination of dependencies confirmed as working for version Alpha 2.73d
- SourceMod 1.12 required
## Requirements
- [SM compiler](https://www.sourcemod.net/downloads.php?branch=1.11-dev):     1.11 - build 6970 (only spcomp and compile files needed)
- [Handles.inc](https://github.com/Isstvan/Slender-Fortress-Reborn/blob/master/compilation/handles.inc):     patched variant (only needed for compile)
- [TF2 Attributes](https://github.com/FlaminSarge/tf2attributes):     1.7.5
- [TF2 Items](https://builds.limetech.io/?p=tf2items):     1.6.4-279 (+ [patched gamedata](https://github.com/asherkin/TF2Items/blob/master/tf2.items.txt))
- [SteamWorks](https://github.com/KyleSanderson/SteamWorks):     git132 (GitHub version, not AlliedModders build - due to dynamic array shenanigans)
- [More Colors](https://github.com/DoctorMcKay/sourcemod-plugins/blob/master/scripting/include/morecolors.inc):     5fd5e7a
- [Native Votes](https://github.com/sapphonie/sourcemod-nativevotes-updated):     6d7b84e (nativevotes_sm_1.12.zip)
- [CollisionHooks](https://github.com/voided/CollisionHook):     1.3.0
- [CBaseNPC](https://github.com/TF2-DMB/CBaseNPC):     1.15.3.123
